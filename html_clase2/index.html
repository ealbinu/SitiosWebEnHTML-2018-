HTML & CSS

Contenido

El HTML es el documento que esta dedicado a la estructuración del contenido de un sitio web.

CSS es el lenguaje que dará estilos al documento web y se trabajo como un archivo separado.

Hoja de Estilos

Las Hojas de Estilo también denominadas CSS son archivos con extensión .css que son linkeados al html mediante la etiqueta LINK

El CSS también puede ser escrito directamente en el HTML en la parte del HEAD o como propiedad style dentro de las etiquetas.

Sintáxis de CSS

            selector {
                propiedad : valor ;
            }
